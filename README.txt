The examples can be run in one of two ways
clicking the run test batch file will run all 5 examples
to run a single example navigate to the examples folder in teh comand line
then put
java -classpath ./ LEDscript < testname.sil
replace test name with the name of the file you wish to run

if any additional examples are written
the batch file will need to be edited to inclued this new example file

the executables files are located in examples folder
